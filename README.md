## BackEnd OverView
This project is the backend of an Employee Management Application developed using:
- Node.js
- Express.js
- MongoDB
- Mongoose

The backend provides REST APIs for managing employee data such as:
- Adding employees
- Fetching employee details
- Updating employee information
- Deleting employees
## FrontEnd OverView
This project is the frontend of an Employee Management Application developed using:
- React JS
- Vite
- React Router DOM
- CSS

The application allows users to:
- View employee details
- Add new employees
- Edit employee information
- Navigate between pages using routing
