# BackEnd OverView
## Technologies Use
- Node.js
- Express.js
- MongoDB
- Mongoose
- CORS
## Backend Project Structure
backend\
│\
├── APIs\
│   └── empApp.js\
│\
├── models\
│   └── EmpModel.js\
│\
├── node_modules\
│\
├── package.json\
├── package-lock.json\
└── server.js\
## Installation Steps
### Step 1: Install Dependencies
```bash
npm install express mongoose mongodb cors
```
### Step 2: Start the Backend Server
```bash
node server.js
```
## To Deploy
### Use Render for backend
### Steps To Deploy
``` bash
 Step 1: Push the file in the vs code to git
 Step 2: connect git to render and select the project
 Step 3: Root Directory : Your Backend File
 Step 4: Select Instance Type : Free
 Step 5: Build Command: npm install
 Step 6 : Start Command : npm start
 Step 7 : Click on deploy
```
